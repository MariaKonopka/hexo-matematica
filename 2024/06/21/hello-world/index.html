Matematicando

Olá! Este blog surgiu como parte das atividades da disciplina de Cálculo Numérico: Métodos para Ajuste de Curvas, oferecida no curso de Ciência da Computação da UniFil. Com o uso do Hexo e do tema IlIke, criei este espaço para funcionar como um portfólio de diversas atividades práticas realizadas em sala de aula. Essas tarefas envolviam a implementação de métodos de ajuste de curvas em código, permitindo que, a partir de determinados inputs, um programa realizasse os cálculos necessários e retornasse o polinômio correspondente ao conjunto de pontos fornecido.

Os códigos

Foram feitos 2 códigos em sala de aula, os métodos de interpolação que foi o de Newton(Diferenças Divididas) e de Lagrange.

Tecnologias

A principio, apenas para fim de estudo do funcionamento dos métodos, os códigos foram escritos em Python utilizando o VsCode.
Uma vez validados e prontos, os códigos foram convertidos para JavaScript e foram aplicados em páginas web construidas com HTML e CSS.

Olá! Este blog surgiu como parte das atividades da disciplina de Cálculo Numérico: Métodos para Ajuste de Curvas, oferecida no curso de Ciência da Computação da UniFil. Os códigos

Uma vez validados e prontos, os códigos foram convertidos para JavaScript e foram aplicados em páginas web construidas com HTML e CSS.

Aplicação do Método de Newton

Aplicação do Método de Newton

No seguinte caso será feito um estudo sobre a aplicação do método de Newton (Diferenças Divididas), onde foi coletado uma folha de árvore qualquer e foi tirado algumas medidas.
Sobre o meio da folha foi traçado um eixo x e tomado pontos de 1 em 1 centímetro, para cada valor de x foi medido do meio à borda folha para achar o valor y, formando assim um par ordenado (x,y).

Os dados estão demonstrado na seguinte tabela:

i x y
1 0 0
2 1.2 0.6
3 2.4 1.5
4 3.1 1.8
5 5 1.6
6 6.1 1.3

Escolhi um ponto a ser interpolado, que no caso foi 1.7, que assim deu uma margem de erro de 0.1.

Método de Lagrange

Método de Lagrange (Interpolação)

Através do método de Lagrange é possível encontrar o polinômio interpolado de menor grau dado um determinado conjunto de pontos. Isso é possível pois cada coeficiente do polinômio é explicitamente escrito através do produto.

Código em JS do método de Lagrange

function calculateInterpolation() {
    const numPoints = parseInt(numPointsInput.value);
    const interpolateAt = parseFloat(document.getElementById('interpolateAt').value);
    const method = methodSelect.value;

    const x = [], y = [];
    for (let i = 0; i < numPoints; i++) {
        x.push(parseFloat(document.getElementById(x${i}).value));
        y.push(parseFloat(document.getElementById(y${i}).value));
    }

    if (method === 'lagrange') {
        const result = lagrangeInterpolation(x, y, interpolateAt);
        document.getElementById('result').innerText = result;
    } else if (method === 'newton') {
        const result = newtonInterpolation(x, y, interpolateAt);
        document.getElementById('result').innerText = result;
    }
}

function lagrangeInterpolation(x, y, value) {
    const n = x.length;
    const steps = document.getElementById('steps');
    steps.innerHTML = '';

    steps.innerHTML += '<p>Valores dos pontos:</p>';
    x.forEach((xi, i) => {
        steps.innerHTML += <p>x${i} = ${xi}, y${i} = ${y[i]}</p>;
    });

    let interpolation = 0;
    let polynomialTerms = `P(${value}) = `;

    x.forEach((xi, i) => {
        let term = y[i];
        let termDetails = ${y[i]};

        x.forEach((xj, j) => {
            if (i !== j) {
                term *= (value - xj) / (xi - xj);
                termDetails += ` * (${value} - ${xj}) / (${xi} - ${xj})`;
            }
        });

        polynomialTerms += ` + ${termDetails}`;
        interpolation += term;
    });

    steps.innerHTML += <p>${polynomialTerms}</p>;
    return interpolation;
}

Interpolação pelo Método de Newton

Método de Newton (Interpolação)

Conhecido como Método de Newton ou Método das Diferenças Divididas, este é um recurso matemático utilizado para gerar um polinômio interpolado a partir de um conjunto de pontos.

Esse método envolve realizar várias operações entre os pontos para determinar a curva que melhor se ajusta ao conjunto de dados fornecido.

Código em JS do método de Newton

function newtonInterpolation(x, y, value) {
    const n = x.length;
    const dividedDifferences = [];
    const steps = document.getElementById('steps');
    steps.innerHTML = '';

    steps.innerHTML += '<p>Valores dos f[x,0]:</p>';
    for (let i = 0; i < n; i++) {
        dividedDifferences.push([y[i]]);
        steps.innerHTML += <p>f[${i},0] = ${y[i]}</p>;
    }

    steps.innerHTML += '<p>Valores dos pontos:</p>';
    x.forEach((xi, i) => {
        steps.innerHTML += <p>x${i} = ${xi}, y${i} = ${y[i]}</p>;
    });

    for (let j = 1; j < n; j++) {
        for (let i = 0; i < n - j; i++) {
            const value = (dividedDifferences[i + 1][j - 1] - dividedDifferences[i][j - 1]) / (x[i + j] - x[i]);
            if (!dividedDifferences[i][j]) {
                dividedDifferences[i].push(value);
            } else {
                dividedDifferences[i][j] = value;
            }
            steps.innerHTML += <p>f[${i},${j}] = (f[${i + 1},${j - 1}] - f[${i},${j - 1}]) / (${x[i + j]} - ${x[i]}) = ${value}</p>;
        }
    }

    let interpolation = dividedDifferences[0][0];
    let polynomialTerms = P(${value}) = ${interpolation};

    for (let i = 1; i < n; i++) {
        let term = dividedDifferences[0][i];
        let termDetails = ${term};

        for (let j = 0; j < i; j++) {
            term *= (value - x[j]);
            termDetails += ` * (${value} - ${x[j]})`;
        }

        polynomialTerms += ` + ${termDetails}`;
        interpolation += term;
    }

    steps.innerHTML += <p>${polynomialTerms}</p>;
    return interpolation;
}
